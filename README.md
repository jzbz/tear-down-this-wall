# Tear Down This Wall!
Paywall circumvention by changing the referrer to Google.

## Supported sites:
* [The Wall Street Journal](http://www.wsj.com/)
* [The Economist](http://www.economist.com/)
* Many more...

## Installation
Tear Down This Wall! is available on the Chrome Web Store:
